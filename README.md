PhishScan Pro
**Advanced Heuristic URL Phishing Detector**

Features
* **Heuristic Scoring Engine:** Real-time risk assessment using weighted algorithms.
* **Infrastructure Check:** Verifies SSL/TLS, MX records, SPF, and DMARC status.
* **Domain Analysis:** Checks for Typosquatting (Homographs), TLD reputation, and age.
* **Modern UI:** Clean, dark-themed dashboard built with Flask.

#### 🛠️ Installation
```bash
git clone https://github.com/YOUR_USERNAME/PhishScan-Pro.git
cd PhishScan-Pro
pip install -r requirements.txt
python app.py
```
